¿Qué quieres aprender hoy?

¿Cómo funciona la web y el Internet?
#web

¿Sabías que la web y el Internet no son lo mismo? En este curso podrás aprender esto y muchas otras cosas
HTML
#web #frontend

HTML es un lenguaje que da estructura a tus páginas web. ¡Sin él, los navegadores no sabrían colocar el contenido de una página o no sabrían diferenciar lo que es un texto de una imagen!
CSS
#web #frontend

CSS es un lenguaje que provee estilos y decoración a tu HTML. Mediante él, puedes convertir un botón soso en un botón impresionante que todo el mundo quiere clicar. También podrás organizar tu página con un diseño más atractivo
JavaScript
#web #frontend

JavaScript es el lenguaje de programación de la web. Lo que se puede hacer en un lenguaje de programación, se puede hacer en JavaScript, con la diferencia de que con JavaScript también puedes manipular HTML y CSS. Con JavaScript, tus páginas ya no son estáticas y obtienen vida
Pensar como un programador
#progamacion

Programar no es simplemente aprenderse la sintaxis de un lenguaje de programación y un par de comandos importantes. En este curso descubrirás que programar es una forma propia de pensar
Python
#progamacion

Python es un lenguaje de programación no solamente útil para quien quiera aprender a programar sin complicarse mucho la vida, sino que también es una herramienta perfecta para quien quiera meterse en el campo de análisis de datos, data science, IoT o incluso la web
Android (con Java)
#android #java

Android es un sistema operativo que no solo se utiliza en smartphones sino también en relojes (smartwatch), tablets, televisores e incluso coches. En este curso de Android aprenderás cómo Android funciona y sabrás desarrollar tus primeras aplicaciones simples usando Java
